# Maison Vale Landing Page

A premium real estate landing page concept for a sample business. The page is built with plain HTML, CSS, and JavaScript and uses remote property photography for a high-end editorial feel.

## Run locally

Open `index.html` in a browser, or serve the folder with a simple static server.

### Optional local server

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive visual design
- `script.js` - navigation toggle and reveal animations

## Publish on GitHub

1. Create a new empty repository on GitHub.
2. Copy the repository URL.
3. From this folder, run:

```powershell
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin main
```

4. In GitHub, open the repository settings.
5. Go to `Pages`.
6. Set `Build and deployment` to `Deploy from a branch`.
7. Choose branch `main` and folder `/ (root)`.
8. Save.
9. Wait for GitHub to publish the site, then open the Pages URL shown in the settings.
