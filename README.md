# DreamNest Realty Sample Website

This repository contains a simple sample real estate business website with:

- Sample property listing data
- Sample SVG property images
- A responsive landing page with listings, agents, and contact details

## Run locally

Because this is a static website, you can open the page directly:

- `index.html`

Or serve it with a local server, for example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
